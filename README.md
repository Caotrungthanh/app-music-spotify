## Hướng dẫn cài đặt:
 * Bước 1: tải project app-music-spotify.
 * Bước 2: mở CMD(Command Prompt) và nhập app-music-spotify hoặc ấn chuột phải vào file mở git bash.
 * Bước 3: mở project lên trình duyệt bằng lệnh npm start.
